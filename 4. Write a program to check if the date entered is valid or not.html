Q. Write a program to check if the date entered is valid or not.



day = int(input("Enter a valid day: "))
month = int(input("Enter a valid month: "))
year = int(input("Enter a valid year: "))
day_in_month = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

if year <= 0 or month <= 0 or day_in_month <= 0:
    print("The Entered Date Is Not Valid.")

if year >= 1 or month <= 12 or day_in_month <= 31:
    print("The Entered Date Is Valid.")
